# Developing this library

## Quick start

* Clone this repo and `cd` into it
* Install npm packages
  * `npm install`

## Running tests

* Install npm packages
  * `npm install`
* Install Java
* Install and start a local DynamoDB instance
  * `grunt local-dynamodb`
* Run the tests
  * `mocha`
